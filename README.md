# Diseño-y-Ejecución-de-Casos-de-Error-en-aplicaciones-Web
● Creación de más de 25 casos de prueba de error para validar flujos críticos como pagos fallidos y errores de geolocalización. 

● Utilización de Figma, JIRA y Excel para el seguimiento y reporte de defectos. 

● Detección y documentación de 10 errores críticos, reduciendo los reportes de falla de los usuarios.
